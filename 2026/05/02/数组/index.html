1. 数组理论知识

定位

  • 存储在连续内存空间的相同数据类型的集合

作用

  • **随机访问: ** 通过索引访问元素快

局限

  • **非末尾增删移动: **连续内存空间存储, 非末尾增删, 需要移动元素

2. 二分

条件

  • 有序序列范围

使用

  • 维护一个答案可能存在的区间
  • 符合条件
  • 区间收敛条件

3. 双指针

使用

  • slow指针

  • fast指针

适用场景

  • 数组
  • 链表

4. 滑动窗口

使用

  • 窗口内容是什么
  • 起始位置移动
  • 末尾位置移动

时间复杂度

为什么时间复杂度是O(n)
不要以为for里放一个while就以为是O(n^2)
主要是看每一个元素被操作的次数
每个元素在滑动窗后进来操作一次，出去操作一次，每个元素都是被操作两次，所以时间复杂度是 2 × n 也就是O(n)。

5. 矩阵

N*N矩阵

  • [螺旋遍历] 圈数 - 每个方向的步长

    • 圈数: n/2

      奇数n, 多出中间坐标(n/2, n/2)
      循环结束后单独处理

    • 方向遍历区间: [start, end)

      左闭右开, 保证每个方向的步长相同

6. 前缀和

背景

  • [遍历求和] 进行n次区间求和, 区间范围为m, 时间复杂度O(n*m)

作用

  • 进行预处理, 区间求和时间复杂度O(1)

  • [本质] 差分算法

使用

  • [定义数组 pre] pre[i]存储0-i的和
  • [获取区间和] 区间和[i,j], pre[j]-pre[i], i为0则为pre[j]

局限

  • 原数组更改, 必须重新创建前缀和

Warning

  • 有序列表, 破坏有序操作
